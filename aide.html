Aide & Utilisation

Guide d'utilisation de l'application API Census - Analyse Démographique Mondiale

← Retour à l'application

🔍 Rechercher un pays

Utilisez la barre de recherche située sur la page principale en saisissant le nom du pays que vous souhaitez analyser (ex: "France"). Cliquez ensuite sur le bouton Rechercher pour interroger l'API et afficher les données.

📊 Comprendre les données

Une fois la recherche aboutie, l'application vous présente une fiche détaillée contenant les informations suivantes :

  • Statistiques générales : Population totale, année de référence, taux de natalité et de mortalité.
  • Estimations : Nombre de naissances et de décès estimés sur l'année.
  • Graphiques et Analyses :
    • Pyramide des Âges pour visualiser la répartition démographique.
    • Évolution de la Population retraçant l'historique et les projections depuis 1950 jusqu'à aujourd'hui.

⭐ Gérer vos favoris

L'application vous permet de sauvegarder vos pays préférés pour y accéder plus rapidement :

  • Ajouter aux favoris : Cliquez sur l'icône favori (étoile/cœur) située à côté du nom du pays dans les résultats de recherche.
  • Consulter : Retrouvez la liste de vos pays sauvegardés dans le panneau latéral gauche (Sidebar) sous la rubrique "Favoris".
  • Supprimer : Retirez un pays de votre liste en cliquant sur l'icône de suppression correspondante dans la barre latérale.
